What NetProbe Does

🌐
DNS Records
Full DNS enumeration — A, AAAA, MX, NS, TXT, CNAME records via Google DNS-over-HTTPS.
passive
📋
WHOIS / RDAP
Domain registration data — registrar, creation/expiry date, nameservers, abuse contacts.
passive
🛡️
Security Headers
HTTP security header audit via Mozilla Observatory — scores CSP, HSTS, X-Frame-Options and more.
passive
🔒
SSL Certificate
TLS/SSL certificate transparency logs via crt.sh — issuer, expiry, SANs, CT log history.
passive
📍
IP Geolocation
Resolve domain to IP, then map to country, ISP, ASN and organisation via ipapi.co.
passive
🔍
Open Ports
Shodan-powered open port discovery — exposed services, banners, known CVEs on the host.
Shodan API
